### PT-BR
Com o arquivo .JPEG dentro desta pasta, é possivel reproduzir uma cópia funcional do dashboard montado para o projeto.

Só é precido ter o aplicativo Blynk no smartphone e scanear o código, desde que se tenha Energy(crédito dentro do aplicativo) suficiente.

With the .JPEG file within this folder, it's possible to repoduce a functional copy of the original dashboard of the project.

You only need to install the Blynk App on you Smartphone and scan the QR Code. You got to have the sufficient amount of Energy (App Currency).
